== Description ==

The WP ShopStype Plugin for Wordpress lets you quickly configure a URL for the ShopStyle API.

It aims to get you up and running as quickly as possible, by giving you a means to generate a URL of lists and/or list items to be imported into a store from the ShopStyle API.

For more details, please see the [ShopStype API webpage]https://www.shopstylecollective.com/api/overview
== Installation ==

To install the store plugin on your site, do the following:

1. In your WordPress Dashboard go to "Plugins > Add New"
2. Click Upload Plugin
3. Browse to the approprioate zip file, then Click "Install Now"
4. Click "Activate Plugin"

That's it, the WP ShopStype should now be installed on your blog.
